## CLI
`python cli.py -h` shows switches for running IDA or Ghidra headless and auto-splitting large text dumps for ChatGPT-sized uploads. Paths to IDA/Ghidra are auto-detected (env, PATH, common install dirs), or override with `--ida-path` / `--ghidra-headless`.
